# Concept of Operations for DRIP Entity Tag (DET) Usage in a Aviation Ecosystems

This is the working area for the IETF Internet-Draft, "Concept of Operations for DRIP Entity Tag (DET) Usage in Aviation Ecosystems".

This document is intended to be submitted to DronesQuad for evaluation to become a Means of Compliance for UAS Remote Identification Session IDs.

## Dependencies & Software

```sh
apt install python3 python3-pip
pip3 install --user rfc2xml
apt install ruby-full
gem install kramdown-rfc2629
```

## Building the Draft

Formatted text, XML and HTML versions of the draft can be built using `make`.

```sh
$ make
```
